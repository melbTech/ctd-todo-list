# Simple Todo Web App

This is a simple todo web app for CTD

### Installation Instructions

Go to GitHub and get the HTTPS and clone it locally using the command:

`git clone [repo https]`

After cloning the repo install the dependencies with the command:

`npm install`

### How to run the development server

Once you have clone it locally and downloaded the necessary files run the program using the command:

`nmp run dev`
